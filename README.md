# Public Subnet Deployment and Remote Access of Windows Server on AWS

## 📌 Project Overview

This project demonstrates how to deploy a Windows Server instance in a public subnet on AWS and access it remotely using Remote Desktop Protocol (RDP).

## 🛠️ Technologies Used

* Amazon Web Services (AWS)
* EC2 (Windows Server)
* VPC
* Remote Desktop Protocol (RDP)

## 🔁 Steps Performed

1. Created a Virtual Private Cloud (VPC)
2. Configured a Public Subnet
3. Attached Internet Gateway
4. Configured Route Table
5. Launched Windows Server EC2 Instance
6. Configured Security Group (Port 3389 for RDP)
7. Connected to the instance using Remote Desktop
